    通过函数正则化的归纳偏置

    通过函数正则化的归纳偏置

    现代机器学习使用大型、高质量的数据集，这些数据集与适当的计算资源一起，推动了具有解释如此大数据的能力的富函数类的设计。这种思维方式在神经网络上很适合，因为即使是最简单的架构，能近似的函数也是一个稠密集。近似几乎任意函数的能力是各种通用近似定理的主题。90年代的应用数学和计算机学出现过一些这样的定理。

    稠密集是这样一个集合，如果一个集合 $A$的闭包满足 $A \cup { \lim_{i\to\infty} a_i : a_i \in A }=X$ 就是说，如果$X$中的任一点$x$可以被$A$中的点任意的逼近。 一个典型的通用近似例子是一个双层感知器， $$f(x) = c\top sign (Ax+b)$$ 在d维度欧几里得空间$R^d$上是稠密的

    然而，通用近似并不意味着_不存在_归纳偏差。给定一个具有通用近似的假设空间，我们可以定义一个复杂性度量$c:F \to R+$并将我们的拟合问题重新定义为 $$\tilde{f} \in \operatorname*{argmin}_{g \in F} c(g) $$ s.t. $$g(x_i) = f(x_i) \space \text{for} \space i=1,..,N$$ 就是说，我们正在我们的假设集中寻找最正则化的函数。对于标准函数空间，这种复杂性度量可以定义为_范数_，从而形成_巴拿赫空间_并允许在泛函分析中利用大量的理论工具。在低维中，样条是函数逼近的主力。它们可以被表述为上面的形式，用一个范数来捕捉经典的平滑概念，例如三次样条函数就可以用二阶导数的范数平方$\int_{-\infty}^{\infty}|f''(x)|^2$

    非正式的说，一个范数$||x||$可以被认为是一个向量$x$的“长度”。 巴拿赫空间是一个定义了范数并对这个范数完备的空间。

    图 1：多层感知器，最简单的前馈神经网络，是通用近似器：只有一个隐藏层，它们可以表示阶跃函数的组合，允许以任意精度逼近任何连续函数。

    在神经网络的情况下，复杂性度量$c$可以用网络权重表示，即$c ( f_\theta ) = c ( \theta )$ 。网络权重的$L_2$范数，称为权重衰减，或所谓的路径范数是深度学习文献中流行的概念。从贝叶斯的角度来看，这种复杂性度量也可以解释为感兴趣函数先验的负对数。更一般地说，这种复杂性可以通过将其放入损失函数来明确表示出来（导致所谓的结构风险最小化），或隐含地，作为某种优化方案的结果。例如，众所周知，欠定最小二乘目标上的梯度下降将选择具有最小$L_2$范数的解。将这种隐式正则化结果扩展到现代神经网络是当前学术研究的主题。总而言之，很自然的，问题出现了：如何定义有效的先验来捕捉现实世界预测任务的预期规律和复杂性？
