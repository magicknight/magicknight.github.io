几何先验

几何先验 现代数据分析是高维学习的代名词。虽然上一节的简单论证揭示了由于维数灾难而无法从通用高维数据中学习，但学习物理结构的数据仍是可能的，我们可以在其中采用两个基本原则：对称性和尺度分离。 在本文中考虑的范围中，这种附加结构通常来自隐藏于输入数据之下的领域：我们将假设我们的机器学习系统对某个域$\Omega$上的信号（函数）进行操作。虽然在许多情况下，$\Omega$上的点的线性组合并没有明确的定义，但我们却可以在其上定义信号的线性组合，即，信号空间形成了一个向量空间。此外，由于我们可以定义信号之间的内积，所以这个空间是一个希尔伯特空间。 要让$\alpha + \beta$有意义，$\Omega$ 必须是个向量空间。 在数学里，希尔伯特空间（英语：Hilbert space）即完备的内积空间，也就是一个带有内积的完备向量空间。这里的完备指：内积空间称为完备的（complete），若其中任意一个柯西序列都收敛。 图 3：如果假设未知函数$f$可以很好地近似为$f ( x ) g ( Ax )$，那么对于$A \in R^{k \times d}$并且$k \ll d$的未知变换，浅层神经网络可以捕获这种归纳偏置。 在典型应用中，这种对低维投影的依赖是不现实的，如本图所示：低通滤波器将输入图像投影到低维子空间。虽然它传递了大部分语义，但同时也丢失了大量信息。 $\Omega$域上的$\mathcal{C}$值信号的空间（$\Omega$是一个集合，可能带有额外的结构，$\mathcal{C}$是一个向量空间，其维度称为_通道_） $$\mathcal{X}(\Omega, \mathcal{C})={x: \Omega \rightarrow \mathcal{C}}$$ 是具有向量空间结构的函数空间。信号的加法和标量乘法定义为： $$(\alpha x+\beta y)(u)=\alpha x(u)+\beta y(u) \quad \text { for all } u \in \Omega$$ 这里的$\alpha$，$\beta$是实数标量。给定$\mathcal{C}$上的内积$\langle v, w\rangle_{\mathcal{C}}$和$\Omega$上的测度$\mu$（有了这两我们可以定义积分），我们可以在$\mathcal{X}(\Omega, \mathcal{C})$上定义一个内积为 $$\langle x, y\rangle=\int_{\Omega}\langle x(u), y(u)\rangle_{\mathcal{C}} \mathrm{d} \mu(u)$$ 作为一个典型的例子，考虑一个二维$n \times n$的网格，$\Omega=\mathbb{Z}{n} \times \mathbb{Z}{n}$，$x$是一个RGB 图像（即信号$x: \Omega \rightarrow \mathbb{R}^{3}$），$f$是一个函数（如单层感知器）作用在$3n^2$维输入上。正如我们将在下面更详细地看到的，域$\Omega$ 通常具有一定的几何结构和对称性。尺度分离原则在这里适用，因为我们在将信号传输到较稀疏版本的域时，能保留信号的重要特征（在我们的示例中，通过稀疏化底层网格对图像进行二次采样）。 我们将看到这两个原则，我们通常将其称为几何先验，在大多数现代深度学习架构中都很重要。在上面考虑的图像的情况下，几何先验以具有共享权重（利用平移对称性）和池化（利用尺度分离）的卷积滤波器的形式构建到卷积神经网络（CNN）中。将这些想法扩展到其他领域，例如图形和流形，并展示几何先验如何从基本原理中出现是几何深度学习的主要目标和我们文本的主旨。 对称性，表示和不变性 非正式地，对象或系统的对称性是使所述对象或系统的某个属性保持不变的变换。这种变换可以是平滑的、连续的或离散的。对称性在许多机器学习任务中无处不在。例如，在计算机视觉中，对象类别不会因位移而改变，因此位移是视觉对象分类问题中的对称性。在计算化学中，独立于空间方向预测分子性质的任务需要旋转不变性. 在描述没有规范排序并且可以任意交换的粒子系统时，离散对称性自然出现，以及在许多动力学系统中，通过时间反演对称性（例如细致平衡系统或牛顿第二定律）。正如我们将在之后的章节中看到的，置换对称性也是图结构数据分析的核心。

维度灾难

维度灾难 虽然低维插值（$d = 1 ,2$ 或 $3$ ）是一个可以计算到很精确误差范围的经典的信号处理任务，可以使用越来越复杂的正则类（例如样条插值、小波、曲波或脊波），高维问题的情况则完全不同。 图 2：我们考虑利普希茨连续函数$f(x) = \int^{2d}_{j=1} z_j \phi(x−x_j)$，其中$z_j = \pm 1$，$x_j \in \mathbb{R}^d$ 对应每个象限，$\phi$在局部上支持利普希茨凹凸。除非我们在$2^d$个象限中的大多数中都有观测，否则我们将在逼近它时产生无法忽略的错误。这个简单的几何论证可以通过为利普希茨函数类定义的_最大差异_的概念形式化。 为了看到问题的本质，让我们考虑一个经典的正则性概念，它可以很容易地扩展到高维：利普希茨连续函数 $f: \mathcal{X} \to \mathbb{R}$ ，即对于所有$x, x’ \in \mathcal{X}$满足$|f(x) - f(x’)| \leq ||x-x’||$的函数。 这个假设只要求目标函数是局部平滑的，即，如果我们稍微扰动输入$x$ （以范数$||x-x’||$衡量），输出$f(x)$不允许有太大的变动。如果我们对目标函数$f$的唯一了解是它是利普希茨连续的，那么我们期望需要多少次观察才能确保我们的估计$\tilde{f}$将接近$f$ ？ 图2揭示了这个问题的答案：需要的观察次数在维度$d$上必然是指数发散的，这表明随着输入维度的增加，利普希茨连续的函数类增长“过快”：在许多具有适中的维度$d$的应用中，需要的样本数量将大于宇宙中原子的数量。甚至如果我们用另一个全局平滑的假设代替利普希茨连续，如索伯列夫空间$\mathcal{H}^s(\Omega_d)$，结果也是一样。事实上，过去的一个研究已经表明$f$的额外平滑假设仅在$s\propto d$时有统计图像上的改善，这在实践中是不切实际的假设。 如果一个函数$f$满足$f \in L^2 (\Omega_d)$，并且广义的$s$阶导数满足平方可积 $$\int |w|^{2s+1} |\hat{f}(w)|^2 dw < \infty$$ 这里 $\hat{f}$ 是 $f$ 的傅里叶变换，$f$就可以被认为在索伯列夫空间中。 图 3：如果假设未知函数$f$可以被很好地近似为$f(x) \approx g(Ax)$，其中$A$是未知的并且$A \in \mathbb{R}^{k \times d}$而且$k \ll d$，那么浅层神经网络可以捕获这种归纳偏置。 在通常的应用中，这种对低维投影的依赖是不现实的，如本例所示：低通滤波器将输入图像投影到低维子空间；虽然它传达了大部分语义，但丢失了大量信息。 完全连接的神经网络定义了一个拥有更灵活的正则性概念的函数空间，这是通过考虑复杂函数$c$的权重来实现的。特别的，通过稀疏性正则化，他们有能力打破这种维度诅咒。然而，这是以对目标函数$f$的性质做出很强的假设为代价的，例如$f$取决于输入的低维投影集合（参见图3）。在大多数现实世界的应用（如计算机视觉、语音分析、物理或化学）中，我们感兴趣的目标函数往往表现出复杂的长程相关性，而这些相关性无法用低维投影来表达（图3），这使得这种假设不切实际。因此，有必要通过利用物理域的空间结构和$f$的几何先验来定义别的正则性，正如我们在接下来的章节中所述。

通过函数正则化的归纳偏置

通过函数正则化的归纳偏置 现代机器学习使用大型、高质量的数据集，这些数据集与适当的计算资源一起，推动了具有解释如此大数据的能力的富函数类的设计。这种思维方式在神经网络上很适合，因为即使是最简单的架构，能近似的函数也是一个稠密集。近似几乎任意函数的能力是各种通用近似定理的主题。90年代的应用数学和计算机学出现过一些这样的定理。 稠密集是这样一个集合，如果一个集合 $A$的闭包满足 $A \cup { \lim_{i\to\infty} a_i : a_i \in A }=X$ 就是说，如果$X$中的任一点$x$可以被$A$中的点任意的逼近。 一个典型的通用近似例子是一个双层感知器， $$f(x) = c\top sign (Ax+b)$$ 在d维度欧几里得空间$R^d$上是稠密的 然而，通用近似并不意味着_不存在_归纳偏差。给定一个具有通用近似的假设空间，我们可以定义一个复杂性度量$c:F \to R+$并将我们的拟合问题重新定义为 $$\tilde{f} \in \operatorname*{argmin}{g \in F} c(g) $$ s.t. $$g(x_i) = f(x_i) \space \text{for} \space i=1,..,N$$ 就是说，我们正在我们的假设集中寻找最正则化的函数。对于标准函数空间，这种复杂性度量可以定义为_范数，从而形成_巴拿赫空间_并允许在泛函分析中利用大量的理论工具。在低维中，样条是函数逼近的主力。它们可以被表述为上面的形式，用一个范数来捕捉经典的平滑概念，例如三次样条函数就可以用二阶导数的范数平方$\int_{-\infty}^{\infty}|f’’(x)|^2$ 非正式的说，一个范数$||x||$可以被认为是一个向量$x$的“长度”。 巴拿赫空间是一个定义了范数并对这个范数完备的空间。 图 1：多层感知器，最简单的前馈神经网络，是通用近似器：只有一个隐藏层，它们可以表示阶跃函数的组合，允许以任意精度逼近任何连续函数。 在神经网络的情况下，复杂性度量$c$可以用网络权重表示，即$c ( f_\theta ) = c ( \theta )$ 。网络权重的$L_2$范数，称为权重衰减，或所谓的路径范数是深度学习文献中流行的概念。从贝叶斯的角度来看，这种复杂性度量也可以解释为感兴趣函数先验的负对数。更一般地说，这种复杂性可以通过将其放入损失函数来明确表示出来（导致所谓的结构风险最小化），或隐含地，作为某种优化方案的结果。例如，众所周知，欠定最小二乘目标上的梯度下降将选择具有最小$L_2$范数的解。将这种隐式正则化结果扩展到现代神经网络是当前学术研究的主题。总而言之，很自然的，问题出现了：如何定义有效的先验来捕捉现实世界预测任务的预期规律和复杂性？
